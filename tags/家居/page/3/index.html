德国发明“贴瓷砖”神器50平方米一个小时搞定，成本竟然只有20元

现在买房子的人越来越多了。 除了买房子，剩下的就是装修房子了。 房子装修后，首先开始装修的地方肯定是地面部分。 这可以说是一个在地上贴瓷砖的大工程。 不仅要找到一个好的主人，还要使用好的水泥。 如果砖坯粘贴不牢或有缝隙，就会出现空心砖，以后修补也会有很多麻烦和浪费。 德国发明了一种贴瓷砖的神器，每小时可以贴50平方米的瓷砖，只需20元。 德国有三个这样的工人，他们一起为别人装饰和粘贴地板。 他们曾经也按照传统的贴砖方法工作，但是效果不理想，工作效率相对较低，所以他们中的一个发明了一种贴砖机，它是由一个两边相对较高的中空方形盒子组成，然后将水泥倒入盒子的中空位置，前面有一个把手，下面有一个齿轮来拉动盒子，慢慢向前移动。 在行走的过程中，水泥可以顺利地通过箱子两侧的高木板铺在地面上。 这种铺水泥的方法不仅平整方便，而且节省了大量的人力，提高了工作效率。 将特别安排一个人在后面贴瓷砖。 这样粘贴的瓷砖光滑结实，速度也提高了。 据工人们说，如果瓷砖上贴有这种艺术品，每小时将贴50平方米的地板，这根本没有问题。 德国人发明的瓷砖神器是用简单的木头和钉子做成的，所以成本只有20元左右，这真是一件物美价廉的好东西。
