银行卡未激活可以转账进去吗？

可以，银行卡没有激活，其他账户仍然可以往这个账户中转账，如果是个人前往银行办理的银行卡，一般都不需要再激活一次，因为在我们办理银行卡时，银行工作人员直接顺带激活了银行卡功能，所以办理完银行卡后都能够正常进行存取款转账等业务办理，如果是由统一组织办理的银行卡则需要用户单独激活。 通常情况下银行卡都要求实名制办理，而有些公司为了方便给员工发工资，会统一给员工办理某家银行的专属银行卡，除此外那些刚刚考完报考完学校的学生们在接收通知书时，通常也会收到由学校为其邮寄的银行卡，这个银行卡也是由学校统一根据学生的身份证信息办理的，银行卡可以用于日后的学费存储、奖金发放以及其他学杂费缴纳功能，而像是这两类由特殊机构统一办理的银行卡都需要使用者在使用之前单独激活。 通常来说只需要用户拿着银行卡钱包所在银行进行密码修改即可。 可虽然说很多人在拿到银行卡时并没有激活，但是在此期间其他账户却可以往这个已经开办的银行账户中打款。 所以这也意味着那些由公司统一办理的银行卡，即便自己还没有办理激活，但是到了发工资时，公司仍然能够正常把钱打入账户中，如果后续用户想要把卡里的钱取出来，则需要携带自己的个人身份证明信息，到银行办理激活手续即可。
