卧室养花，这“3种”别养，太香了，会跟你争夺氧气，天天失眠

我们在卧室养花，最好养一些比较清新，雅致的盆景盆栽。 因为卧室通风条件比较差，养护植物最好养，耐阴好养的观叶植物。 如果是想养花的话，不要养香气太重的花卉，容易跟人争夺氧气，提高人的换气频率，而且对于睡眠不太好。 容易引起失眠。 那么适合在卧室养殖的盆栽植物有哪些？ 平时我们养护的大部分耐阴的观叶植物比如绿萝、吊兰、龟背竹、玉树、芦荟、虎皮兰等等都可以在卧室中养护。 养护一些绿植叶可以净化卧室的空气，释放氧气，对人的身体健康比较有好处。 但是对于一些开花的植物比如说瑞香、水仙、风信子等等香气过浓的植物最好不要放在卧室。 在卧室养花平时要多开开窗通通风，让卧室的空气流通一下，有利于身体的健康，也有利于盆栽花卉的呼吸作用，更好的进行新陈代谢。 养护这些盆栽花卉不易长期在背阴的地方，可以隔段时间拿到光照充足的地方晒一晒。 或者是几种植物交替放在卧室中养护。 会给我们增添了新鲜感，也有利于植物的生长。 卧室中的环境还需我们经常清扫打理。 对于盆栽花卉也要及时的清除黄叶烂叶，擦拭叶片。 那卧室看起来更加整洁清新。 卧室养花，这“3种”别养，太香了，会跟你争夺氧气，天天失眠，卧室养花，切记这2点，你都知道吗。

石碌含笑，“香”得很，一点也不娇气

石碌含笑，和峨眉含笑同属木兰科含笑属，都是国宝级别的。 它们的花一白一黄，在大自然相映成辉。 石碌含笑就是名字“土”了点，但它一下就能让我们知道它的“籍贯”：海南昌江石碌镇。 喜温暖湿润气候，喜光耐干旱瘠薄稍耐阴，对土壤要求不严，全日照半日照都可以。 自然生长总是喜欢在山沟、山坡、路旁、水边。 石碌含笑的花是很美的。 花被片9枚，3软。 每到春夏开花时节，开出洁白的花朵，苍润如玉，浓香扑鼻。 它的顶芽带橙黄色或灰色金属色，光彩夺目，静雅美丽。 石碌含笑是小乔木，高可达18米，胸径30厘米，树干通直，绿叶葱笼，分枝上举，树冠呈优美的塔形。 种类也很多，株形大小高矮不一。 如果说让我来推荐南方景观树种，我当首推石碌含笑。 它不娇气，花儿美而香，种类株形选择还多。 而事实上，它在实际运用中“香”得不行，已广泛种植于公园、景观区、花坛、行道树等等园林绿化中。 或单植，或列植，或群植，为人类美好生活增色添彩。
