岁月催人老，但人的内在个性，永远不会老

泰戈尔说：“当青春的光彩渐渐消逝，永不衰老的内在个性，却在一个人的脸上和眼睛上，更加明显地表露出来，好像是在同一地方久住了的结果。 ” 人生，就是一个由生长，到衰老的过程。 童年，少年，青年，中年，老年，人生的轨迹，清晰可见，谁都无法更改。 在人的成长过程中，人的外表，会随着年龄的增长，而不断地走向衰老。 而内在的个性，修养，精神，气质，却会随着年龄的增长，而不断地积蓄，不断地加深，不断地提高，最终形成了人生的魅力。 岁月的风霜，只能染白人们的双鬓，却不能使人们的生活变得苍白，中老年人生的魅力就是这样形成的。 人生的不同阶段，自有不同的特点和风采，我们不要惧怕，我们不要回避，我们要乐观地看待人生，我们要勇敢地展现自己。 人的生老病死，是自然规律，不断地成熟和不断地完美，才是我们应该做的事情，有自信的人，才会有更加亮丽的人生。
