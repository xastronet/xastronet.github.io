那些“残次品”CPU，最后都去了那里？

半导体生产属于高精度的生产工艺，然而良品率并不会很高，经常产生一些不合规的“残次品”，为了减少浪费，本着“又不是不能用”的原则，被厂家做成各种“次一级产品”。 “残次品”CPU因为在标准规格下运行会出现过热、能耗过高的不稳定情况，所以一般会采取锁核、降频等手段进行”改良“于是就产生了一些其他型号的产品。 比如苹果a15的降频版本就用在2021iPad mini上，从3.22Ghz降频到2.93Ghz，让这些”体质不好“的cpu也能稳定运行，达到一个性能和功耗都不错的体验。 也有一些PC的cpu，根据体质加上不同后缀来区分型号，各位可以通过gpu gflops、或者CPU-Z查看一下自己的CPU体质，体质优秀的可以长时间开启超频。 最近AMD还用索尼PS5”瑕疵品“芯片捣鼓出一台4700S的迷你主机套件，基于Zen2架构，8核心16线程，二级缓存4MB，三级缓存8MB，最高频率仅3.2GHz，并且自带了自带16GB GDDR6内存，还可以加装Radeon 550、RX 550等显卡，目前AMD透露已经有原始设备制造商基于这个套件生产超过80款设备。 无可厚非这都是对”残次品“CPU物尽其用的手段，根据品质区分型号比重新设计不同规格的处理器要更加高效，特别是在今年半导体紧缺的情况下，我们需要更多性价比高的处理器“满足需求。 那么各位机友们，你会选择”高品质“还是”高性价比呢“？

把电脑运行内存升级到1TB，大概需要多少钱？

普通家用电脑或者说是工作站电脑目前都不可能达到1T内存，1TB=1024GB，然而目前ddr4内存单条容量仍然不够大，单条32G内存非常稀少，价格也很昂贵， 主流ddr4内存还是以单条8-16G为主；另外，大部分主板最多只有8根内存插槽，在现有的单条内存容量下不计成本使用8*32G内存也不过256G的内存容量。 所以如果想使用超过1T的内存只有使用专业级服务器或者超级计算机，通过多台机器并联从而达到超过1T内存的电脑，如果你在使用64位win10系统的话，家庭版最多只能支持128G内存容量，只有企业版和专业版能够支持到最大2T内存容量。 目前三星单条32G内存价格在2500-3000元左右，如果按照2500元一条的价格来算，用这样的内存组成1TB内存规格需要80000元之多，已经非常吓人了，这么大的内存只有在高性能计算和服务器领域才能用得到。 这对于家用娱乐电脑来说，16G内存已经可以满足99%的需求了，未来几年内32G仍然属于少数高端用户的配置。

DDR5内存都爱强调 内存延迟为啥那么重要

12代酷睿马上正式上市，它的两个好搭档Z690主板和DDR5内存也已经露面，甚至上市了。 细看各大厂商的发布信息就会发现，几乎所有的DDR5内存在发布时都会在频率之后挂上一串数值，也就是所谓的延迟参数，它到底是啥？ 顾名思义，所谓的延迟就是内存运行时需要多久才能有所反应，其中最重要的就是一般被列在第一个的CL值，全称为CAS（Column Address Strobe）延迟时间，内存接收到一条指令后要等待多少个时钟周期才能执行任务，它的值越小，内存对数据的反应速度越快，性能越好。 按照一般的标注顺序，第二个数是RAS(Row Address Strobe)，表示内存行地址传输到列地址的延迟时间；第三个数为RAS Prechiarge（内存行地址脉冲预充电时间）；下面还有Act-to-Prechiarge（内存行地址选择延迟）。 那么，为啥厂商特别重视延迟参数呢？ 原因说来也简单，那就是厂商也不指望用DDR5和DDR4相比，而是和其他产品相比，比如常见的DDR5 4800的CL值是40，那么CL为36的DDR5 4800甚至更高频率内存肯定会引人瞩目。 至于为啥DDR5厂商突然开始特别爱强调延迟，大概还一个原因就是之前曾屡屡出现高频内存隐藏延迟数据，造成某些情况下性能比主流频率内存更低的情况。 对于起始频率就很高的DDR5内存，索性说明白也是让消费者放心的手段。

拥有一身正气的南方人，是如何挑选取暖器的？

作为一个身处南方的住户，集中供暖必然与我无关。 因此过冬的方式除了出门靠一身正气以外，屋内就得依靠各种取暖器了。 小太阳、油汀、暖风机、地暖、浴霸、踢脚线取暖器… 那么除了空调和地暖以外，该挑选什么取暖器呢？ 我个人觉得，在挑选前一定得明确自身的需求，是要在客厅、卧室还是浴室中使用，是要局部取暖还是整体升温？ 假如是用到客厅中，那么也可用快热炉、踢脚线或远红外取暖器。 取暖器之间的区别是什么？ 而红外式取暖器则以小太阳取暖器、远红外式取暖器，也就是散发着红光的。 油汀取暖器：工作原理是先加热导热油，在腔体内形成内对流，然后再通过散热片将热量散发至空气中。 目前油汀的散热片数量在11至15片之间，普通13片机型较多。 其中可见光的波长在0.39μm至0.77μm范围间，不可见光的范围在0.76μm至1000μm。 接下来，就是关于刚入手的舒乐氏（SOLEUSAIR）远红外碳素管取暖器的体验过程， 舒乐氏远红外碳素管取暖器（以下简称舒乐氏远红外取暖器）到手时是双层包装箱，而且某东物流都是八角尖尖，外观没有任何问题。 不过幸好舒乐氏远红外取暖器带有过热自断电保护，可以有效避免险情的发生。 当我们将舒乐氏远红外取暖器放倒时，它会及时自动断电，立起后才会再次恢复工作，这对于取暖器来说，是很重要的安全保护。 如需迅速升级，那么远红外取暖器会更好些。
